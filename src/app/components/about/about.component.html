Este proyecto fue creado como parte de un ejercicio práctico de los principios básicos sobre la Comunicación entre componentes, Gestión de eventos Formularios, PWA y despliegue en un servidor remoto, Almacenamiento de datos temporales y autenticacion mediante el uso de Firebase, guiado por el profesor JORGE ARTURO SAUCEDA FLORES.
